# NOVA-FCT-VPN-GUIDE
Go [Here](https://github.com/ancwrd1/snx-rs/releases) and download the installation files that fit your system.<br>

You can either run this setup installation via python or bash, whichever you prefer.<br>
After downloading simply run the `setup.py` file in the folder `Python_Version` or `setup.sh` in the folder `Bash_Version`
and it'll automatically create the bash script `vpn.sh` in your home folder that starts the vpn, as well as a desktop 
shortcut called VPN for it.<br>

## Footnote
This setup only sets up an easy way to start up the vpn on your machine, but the development
of the vpn client itself is not maintained here clearly. For information and news please visit it's repository.
