# NOVA-FCT-VPN-GUIDE
Go [Here](https://github.com/ancwrd1/snx-rs/releases) and download the installation files that fit your system.

After installing simply run the `setup.py` file and it'll automatically create the bash script `vpn.sh` in your home folder
that starts the vpn, as well as a desktop shortcut called VPN for it.<br>

## Compatibility
This setup should work on major linux distros such as Mint, Ubuntu, Debian, Arch, Fedora.
All debugging would be straightfoward due to the simplicity of this script.
For other distros compatibility is NOT assured.

## Footnote
This setup only sets up an easy way to start up the vpn on your machine, but the development
of the vpn client itself is not maintained here clearly. For information and news please visit it's repository.
